Mistake Analysis

Find Danger Zones Before the Exam

Logistic regression computes P(miss) per student per topic. Co-failure patterns reveal hidden weaknesses.

⚠️

P(miss) Per Topic

Logistic regression trained on each student's attempt history predicts the probability of getting a topic wrong on the next exam.

🔗

Co-failure Patterns

Discovers topic pairs that students consistently get wrong together — revealing conceptual gaps that span chapters.

🚨

At-risk Flagging

Automatically flags students whose danger-zone count exceeds thresholds, surfacing them on the org dashboard for intervention.

Model: amanrr48/prajna-mistake-predictor on HuggingFace · Per-student logistic regression with topic-level features
